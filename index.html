📋 변경 이력 (Release Notes)
v3.2.0 — 2026.04.12
  • 국내 지수 엔진 고도화: 네이버 금융 실시간 API 연동 및 하이브리드 수집 로직으로 코스피/코스닥 지수 안정적 복구
  • 차트 연동 최적화: 지수 클릭 시 ETF(SPY 등)가 아닌 실제 지수(SPX, NDX, DJI) 트레이딩뷰 차트 호출
  • 초기 로딩 로직 개선: 끊어진 초기화 체인을 복구하여 대시보드 진입 시 즉각적인 데이터 렌더링 보장
v3.1.0 — 2026.04.12
  • 배당 차트 추정 모델 도입: 과거 이력 합산 방식에서 '연간 기대치 기반 미래 스케줄 분배' 모델로 전환
  • 4월 배당 왜곡 해결: 데이터 수집 경계선 문제로 특정 달의 배당금이 낮게 표시되던 현상 완벽 수정
  • 기능 복구 및 안정화: 모바일 헤더 레이아웃 최적화 및 롤백을 통한 사용성 회복
v3.0.0 — 2026.04.12
  • 하이브리드 로딩 (v3.0): 야후 Batch API + 백그라운드 배당 분석으로 로딩 속도 비약적 향상
  • SWR 캐싱 도적: 이전 세션 시세 즉시 복원 → 0초 렌더링 후 백그라운드 갱신
  • 모바일 카드 UI 개편: 3열 그리드 배치 및 현재가/등락률 우측 밀착 정렬로 정보 밀도 강화
  • 실시간 통화 연동: 통합 통화 토글 시 개별 종목의 평가액/손익도 실시간 환산 반영
  • 통신 안정성 강화: 프록시 타임아웃(4초) 및 개별 종목 폴백 알고리즘 적용
v2.5.0 — 2026.04.11
  • 차트 분리: 인라인 차트 제거 → 태커/지수 클릭 시 새 탭에서 TradingView/StockPlus 열기
  • 배당률 컬럼: 종목 테이블에 YOC(투자배당률) + 납입월 + 연간 배당액 표시
  • 15.4% 세금 토글: 배당 구성 탭에 세금 적용 ON/OFF 버튼 추가
  • YieldMax 지원: 가격 fetch 시 dividendYield 동시 저장 → 별도 API 없이 배당률 표시
  • us. / kr. 접두사 처리: 티커 정규화 함수로 Yahoo Finance 호환
v2.4.0 — 2026.04.11
  • 자산 구성 도넛 차트: SVG 도넛 차트 + 2열 레전드 (비중% 표시)
  • 배당 구성 탭: 연간 배당금 / 투자 배당률(YOC) / 시가 배당률 카드
  • 월별 배당 바차트: 실제 이벤트 기반 + 분기 추정 폴백 (현재월 강조)
  • 한국 주식 배당: 숫자 티커(441640 등) 자동 감지 → .KS 추가 조회
  • 배당 지연 로드: 가격 조회와 충돌 방지 (5초 지연 / 탭 클릭 시 즉시)
v2.3.0 — 2026.04.11
  • 연간 배당 예상 카드: 포트폴리오 총 배당금 + 수익률 요약
  • 미국 10년물 국채 금리: ^TNX 지수바 신규 추가 (TVC:US10Y 연동)
  • CNN 공포탐욕지수: VIX 옆 배치, 다중 프록시 폴백(직접→allorigins→corsproxy)
v2.2.0 — 2026.04.11
  • 지수바 재구성: S&P500, 나스닥, 다우, VIX, 환율, 금, 은, WTI + 국채
  • 다중 프록시: allorigins.win → corsproxy.io 자동 폴백
  • 모바일 최적화: 375px 뷰포트 기준 종목 카드 레이아웃 개선
v2.1.0
  • USD/KRW 토글: 전체 화면 통화 전환 (자동 환산)
  • 계좌별 탭: 계좌 등록 및 탭 분리 보기
  • 수동 가격 입력: 조회 불가 종목 수동 가격 지정
  • 데이터 내보내기/불러오기: JSON 백업 및 복원
v2.0.0
  • Yahoo Finance API: 실시간 가격 조회 (CORS 프록시 경유)
  • FMP API 지원: Financial Modeling Prep 대체 데이터 소스
  • TradingView 연동: 종목/지수 클릭 → 차트 (현재는 새 탭)
  • 다크 모드 UI: Glassmorphism 기반 프리미엄 디자인
